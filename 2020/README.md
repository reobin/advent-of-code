# AoC 2020

## Getting started

### Requirements

* Go version 1.15

### Run the solution for a specific day

```shell
bin/start 01 # Runs day 01
bin/start 24 # Runs day 24
```

### Run tests

```shell
bin/test # Runs all go test files
```

### Run linter

```shell
bin/lint # Prints lint errors to fix
```

### Run code formatter

```shell
bin/format # Formats and save all go files
```
